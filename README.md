# Jogo da Cobrinha

Este é um jogo simples da cobrinha feito com HTML, CSS e JavaScript.

## Como Jogar

- Use as setas do teclado para controlar a direção da cobrinha.
- Tente comer as maçãs (quadrados vermelhos) para crescer.

## Visualize o Jogo
